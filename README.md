# Sluice — Bug Reports & Feature Requests

This is the public issue tracker for [Sluice](https://sluice.sh), the monitoring and alerting platform for Celery job queues.

## How to report a bug

[Open a bug report →](https://github.com/sluicehq/sluice/issues/new?template=bug_report.yml)

Include:
- What happened vs what you expected
- Steps to reproduce
- SDK/Agent version and Celery version
- Relevant logs (redact any sensitive data)

## How to request a feature

[Open a feature request →](https://github.com/sluicehq/sluice/issues/new?template=feature_request.yml)

## In-app feedback

You can also submit feedback directly from the Sluice dashboard — click the **Feedback** tab on any page, or press `Cmd+K` and search "feedback."
